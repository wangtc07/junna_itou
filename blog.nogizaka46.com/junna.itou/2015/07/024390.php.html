2015/0714Tue伊藤純奈♯75個目 チョコレ ートミント味

 
 
こんばんわんこ！
伊藤純奈です
 
 
のぎるーむで着てたパジャマ♡
gelatopiqueのものだよ〜
 
 
 
 
 
 
 
 
 
こんぺいとーく！
 
 
 
☆美容院にはどの位で行ってるの？
 
→ うーん、3週間に1回とかかなぁ。
 
 
☆この夏、どこか旅行できるとしたら
   どこに行きたい？
 
→ 海外！綺麗な海で遊びたい！
    夏は全国ツアーがあるので
    落ち着いたら家族で旅行したいなぁー
 
 
 
☆純奈が乃木坂の握手会にファンとして
   行くなら誰のレーンに行ってみたい？
 
→ この話よく伊織とするんだけど、
    結果選べないよねってなる。笑
 
 
☆好きなジブリ作品どれかひとつでも
   良かったら教えて
 
→ ハウルの動く城
 
 
☆生まれ変わるならーーー？
 
→ 足の速い男の子
 
 
 
☆握手会で何話せばいい？
  なんでもいいよは無しな
 
→ 皆さんが券を買ってくれてわざわざ
    来て下さってるので皆さんが
    話したいと思った事を話したいですっ
    
 
 
 
☆お仕事とかで色んな髪型すると
  思うけど純奈的には何が1番好き？
 
→ 触覚だしポニーテール
 
 
 
 
 
☆純奈から見て騎馬戦ってどんなイメージ？？
 
→ こわい。やりたくない。こわい。笑
 
 
 
 
☆握手会でいじられるのってどうなの？
   俺の場合は愛情のあるいじりをしてる
   つもりなんだけど...
   そんで純奈たんも凄く楽しそうだった
   からさ、ついつい...
 
→ 愛のあるいじりは楽しいし面白いけど
    度が過ぎると、あーーーってなる時も
    ありますね( ^ω^ )
 
 
☆地球最後の日 最後に何が食べたい？
 
→ 焼肉
 
 
☆寝る前に必ずやる事ある？
 
→ ストレッチ
 
 
☆1週間の内好きな曜日は？
 
→ 金曜日と土曜日
 
 
 
☆最近のマイブームは？
 
→ タピオカ
 
 
 
☆小・中・高で色んな夏休みの課題が
  あったと思うけど一番苦手だったなぁー
  っていう課題は？
 
→ ワークとかドリルとかそこらへん
 
 
☆純奈の世界史などの暗記方法教えてー？
   ダンスとかも覚えるの大変そうだよね
 
→ 青ペンでひたすら書く。
    体動かしながら声に出して覚える。
    
 
 
 
 
今日はここまで！
 
 
 
 
 
 
 
 
 
 
 
 
 
 
この前まで期末テストが
ありました！この前の中間より
よく出来たからハッピー！
数学は前回より点数上がったから
先生に褒められて嬉しかった( ´ ▽ ` )ﾉ
勉強と仕事の両立頑張るぞー！
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
◎初森ベマーズが始まりました！
1話はしっかりリアルタイムで
見たよー♪♪
予告で純のシーンがあったので
どんなビジュアルで出るかは皆さん
ご存知かと思いますが......。笑
楽しみにしててください(((o(*ﾟ▽ﾟ*)o)))
 
 
◎NOGIBINGO!5が始まりました！
4に引き続き5もやらせて頂けて
本当に嬉しいですm(._.)m♡
来週も出るのでチェック宜しくです！
 
そして、NOGIBINGO!5初回放送で
告知されてましたが琴子、絢音、蘭世、
みり愛との5人で妄想リクエスト出場権
をかけて再生回数対決をします！動画は
huluで配信されます！先日その撮影を
してきたんだけど本当に恥ずかしくて
恥ずかしくて....。笑
1番多く再生された人が妄想リクエストに
出れるので皆さんぜひ！
応援よろしくお願いします( ^ω^ )
 
 
◎ヤングアニマル
アンダーメンバーで表紙、巻頭、巻末
をやらせて頂きました〜
7月24日発売です！
 
 
◎BUBKA9月号
塩アイス(北野、堀、純奈)で
取材して頂きました！ずっと
やりたかったから本当に嬉しかった♡
7月31日発売です！
 
 
◎12枚目シングルtypeAに入る
蘭世とのペアPVの予告編が
公開されてます！オフショットは
発売されてから載せますね♪
 
 
◎のぎ天ゴルフ部配信されてます！
ぜひみてくださーいm(._.)m
 
 
◎のぎるーむもhuluで配信されてる
ので見てください♡
まあやさん、かりん、みおな、日奈子
の5人でやりました〜楽しかった！
 
 
 
 
 
あ！7月お誕生日の方は
コメントよろしくです( ´ ▽ ` )ﾉ
 
 
 
 
 
 
それでは、
おやすみなさーい！
