2015/0727Mon伊藤純奈♯77個目 傷だらけ のローラ

 
 
 
 
こんばんわんこ！
伊藤純奈です
 
 
 
 
 
 
初森ベマーズ3話
見ていただけたでしょうか！
 
ベマーズの敵チーム「日京女子学院」の
傷だらけのローラ役で出させて
頂きました♪♪日京女子学院は
トライバルのレイ役らりん姉さん、
金髪のエリー役まひろさんの3人です！
 
1話の予告編に出てたので1話に
出ると思ってた方すみませんm(._.)m
飛び蹴りなんて強烈なシーンだったから
CM流れるたびに本当に
恥ずかしかったよ....。笑
 
 
逆になっちゃってるけど！
 
 
ヤンキー役ということで、
真っ赤なリップに跳ね上げライン
髪型はソバージュのような感じで
コテコテのヤンキーになりました。笑
 
 
 
 
名前の通り傷だらけにも
なったんだよ〜。腕にも顔にも！
帽子も純のだけざっくざくに
傷がついてた(((o(*ﾟ▽ﾟ*)o)))
 
実はこの後、傷の跡で焼けちゃって
腕に本当の傷の跡みたいなのが
出来てたの。恥ずかしい。笑
 
 
 
 
トライバルのレイ役のらりん姉さん♡
 
 
 
 
 
 
これからどんどんメンバーが
出てくるので見逃さないでね( ´ ▽ ` )ﾉ
 
 
 
 
 
握手会の事はまた次回
書きます！！
 
 
それではっ
おやすみなさい！

2015/0714Tue伊藤純奈♯75個目 チョコレ ートミント味

 
 
こんばんわんこ！
伊藤純奈です
 
 
のぎるーむで着てたパジャマ♡
gelatopiqueのものだよ〜
 
 
 
 
 
 
 
 
 
こんぺいとーく！
 
 
 
☆美容院にはどの位で行ってるの？
 
→ うーん、3週間に1回とかかなぁ。
 
 
☆この夏、どこか旅行できるとしたら
   どこに行きたい？
 
→ 海外！綺麗な海で遊びたい！
    夏は全国ツアーがあるので
    落ち着いたら家族で旅行したいなぁー
 
 
 
☆純奈が乃木坂の握手会にファンとして
   行くなら誰のレーンに行ってみたい？
 
→ この話よく伊織とするんだけど、
    結果選べないよねってなる。笑
 
 
☆好きなジブリ作品どれかひとつでも
   良かったら教えて
 
→ ハウルの動く城
 
 
☆生まれ変わるならーーー？
 
→ 足の速い男の子
 
 
 
☆握手会で何話せばいい？
  なんでもいいよは無しな
 
→ 皆さんが券を買ってくれてわざわざ
    来て下さってるので皆さんが
    話したいと思った事を話したいですっ
    
 
 
 
☆お仕事とかで色んな髪型すると
  思うけど純奈的には何が1番好き？
 
→ 触覚だしポニーテール
 
 
 
 
 
☆純奈から見て騎馬戦ってどんなイメージ？？
 
→ こわい。やりたくない。こわい。笑
 
 
 
 
☆握手会でいじられるのってどうなの？
   俺の場合は愛情のあるいじりをしてる
   つもりなんだけど...
   そんで純奈たんも凄く楽しそうだった
   からさ、ついつい...
 
→ 愛のあるいじりは楽しいし面白いけど
    度が過ぎると、あーーーってなる時も
    ありますね( ^ω^ )
 
 
☆地球最後の日 最後に何が食べたい？
 
→ 焼肉
 
 
☆寝る前に必ずやる事ある？
 
→ ストレッチ
 
 
☆1週間の内好きな曜日は？
 
→ 金曜日と土曜日
 
 
 
☆最近のマイブームは？
 
→ タピオカ
 
 
 
☆小・中・高で色んな夏休みの課題が
  あったと思うけど一番苦手だったなぁー
  っていう課題は？
 
→ ワークとかドリルとかそこらへん
 
 
☆純奈の世界史などの暗記方法教えてー？
   ダンスとかも覚えるの大変そうだよね
 
→ 青ペンでひたすら書く。
    体動かしながら声に出して覚える。
